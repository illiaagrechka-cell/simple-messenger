# Простой мессенджер

Минимальное, но полностью рабочее веб-приложение: регистрация по имени и паролю, отправка сообщений другим зарегистрированным пользователям. Данные хранятся в файлах на диске (`data/users.json`, `data/messages.json`), поэтому переписка сохраняется между перезапусками сервера.

## Как запустить

Нужен установленный [Node.js](https://nodejs.org) (версия 18 или новее).

1. Откройте терминал в этой папке.
2. Установите зависимости (один раз):
   ```
   npm install
   ```
3. Запустите сервер:
   ```
   npm start
   ```
4. Откройте в браузере: **http://localhost:3000**

## Как переписываться с другим человеком

Приложение работает по сети — если запустить его на одном компьютере, к нему смогут подключиться и другие устройства в той же сети (или в интернете, если открыть порт/задеплоить на хостинг).

- Простой вариант: оба человека открывают одно и то же приложение через один и тот же адрес (например, вы запускаете сервер и даёте собеседнику ссылку вида `http://ВАШ-IP:3000`, либо оба заходят на `http://localhost:3000` с одного компьютера в разных вкладках/браузерах для теста).
- Каждый регистрируется под своим именем и паролем, затем вводит имя собеседника в поле "имя собеседника" и пишет сообщение.
- Список уже зарегистрированных пользователей виден на главном экране — можно просто нажать на имя, чтобы открыть чат.

## Важно

- Это простой учебный проект, а не продакшен-система: пароли хранятся в захэшированном виде, но нет HTTPS, ограничений скорости запросов и других механизмов защиты.
- Сессии (вход) хранятся в памяти сервера — при перезапуске сервера все пользователи будут разлогинены (данные пользователей и сообщений при этом не теряются).
- Чтобы приложение было доступно из интернета (не только в локальной сети), его нужно разместить на хостинге (например, Render, Railway, VPS и т.п.) — это можно сделать по желанию отдельным шагом.
